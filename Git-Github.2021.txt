Funcionalidades
Branches - Ramificações do código

Trabalho em equipe em tempo real - cada um trabalha no seu “galho” e o repositório “junta as peças”.
Precisa adicionar ou alterar algo no código mas não tem certeza se é uma boa idéia? Os branches podem ser modificados sem interferir no código principal.
Pode ser local ou remoto - mesmo em equipe, só você modifica sua parte. 




Funcionalidades
Versionamento.

 Centralizado - todos os commits “se juntam” em um ponto só. 
Distribuido - são feitas várias versões diferentes que são salvas por completo e “não mescladas”.
Permite que se continue o trabalho em máquinas variadas.
